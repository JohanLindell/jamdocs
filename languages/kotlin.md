## Do while loop
```kotlin
do {
  // code block to be executed
}
while (condition);
```

## For loop
```kotlin
 for (var i = 0; i < items.length; i++) {
   items[i] // use me
 }

for (var i in range(0..items.length)) {
  items[i] // use me
}
```

## For each loop
```kotlin
for (variable in iterable) {
  variable // use me
}
```

## Ternary
```kotlin
if (condition) trueExpression else falseExpression
```

## While loop
```kotlin
while (condition) {
  // code block to be executed
```
