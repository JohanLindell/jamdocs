## Do while loop
```javascript
do {
  // code block to be executed
}
while (condition);
```

## For loop
```javascript
var i;
for (i = 0; i < items.length; i++) {
  items[i] // use me
}
```

## For each loop
```javascript
for (variable of iterable) {
  variable // use me
}
```

## Ternary
```javascript
condition ? ifTrueExpression : ifFalseExpression
```

## While loop
```javascript
while (condition) {
  // code block to be executed
}
```
